## Integração com o Projeto Java da Disciplina

Aplicação direta no projeto base:

1. Introdução de entidades com invariantes explícitas.
2. Separação domínio/orquestração.
3. Redução de acoplamento via interfaces.
4. Uso de testes como congelamento comportamental.

Estrutura recomendada:

- domain/
- application/
- infrastructure/
- presentation/

Alinhamento com arquitetura limpa (Martin, 2017).
